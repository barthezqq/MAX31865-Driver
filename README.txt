*安装指南*
**********
名称：MAX31865 示例工程
版本：19-05-07    Rev 1.0    
      Maxim Integrated TTS Asia New-release.

主要功能： 利用SPI通信协议控制MAX31865数字转换器完成RTD测温。

硬件：例程基于MAX32625/6 EV kit REV 2.0评估板开发，运行时总线上连接MAX31865PMB1。
      评估板链接：
      https://www.maximintegrated.com/cn/products/microcontrollers/MAX32625-EVKIT.html

软件：
      开发环境：Keil_v5, MDK Version 5.24.1

运行指南：
      Step1： 安装MDK
      Step2： 选择 Pack Installer-> 左上角刷新器件列表,等待更新
      Step3： 在Device中选择[Maxim]->[MAX32625]->[MAX32625]
      Step4： 选择[Device Specific]->Maxim::MAX32625 选择下载最新版支持包。
      支持包下载完成后运行本工程即可。

技术支持：
      https://maximsupport.microsoftcrmportals.com/zh-CN/support-center/
